# 🚀 Python Projects Collection

## 📌 Overview

This repository contains three Python projects developed to improve programming, problem-solving, GUI development, file handling, and application design skills.

Each project focuses on a different domain:

- 📈 Finance Management
- 🎮 Game Development
- 🏥 Healthcare Assistance

---

## 📂 Projects Included

### 1. Stock Portfolio Tracker 📈

A modern GUI-based application that helps users manage and monitor stock investments.

#### Features
- Add and remove stocks
- Portfolio tracking
- Profit/Loss calculation
- Portfolio allocation analysis
- CSV save and load functionality
- Export portfolio reports
- Pie chart visualization
- Interactive dashboard

#### Technologies Used
- Python
- Tkinter
- ttkbootstrap
- Matplotlib
- CSV

---

### 2. Hangman Game 🎮

A classic word-guessing game developed using Python.

#### Features
- Random word generation
- Multiple guessing attempts
- Input validation
- Win/Loss detection
- Interactive gameplay
- Console-based user interface

#### Technologies Used
- Python
- Random Module

---

### 3. Healthcare Reminder Chatbot 🏥

A healthcare assistant chatbot designed to help users manage daily health-related activities.

#### Features
- Medicine reminders
- Health tips and suggestions
- Appointment reminders
- Interactive chatbot conversation
- User-friendly interface

#### Technologies Used
- Python
- Tkinter
- Datetime Module
