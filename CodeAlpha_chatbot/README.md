# 🌿 HealthBot - Personal Wellness Companion

A modern desktop healthcare assistant built with **Python** and **Tkinter** that provides wellness guidance, disease information, BMI calculation, emergency contacts, and personalized health tracking.

---

## 📌 Overview

HealthBot is an interactive healthcare and wellness application designed to help users access basic health information, healthy lifestyle recommendations, emergency contacts, and wellness tools through a user-friendly graphical interface.

---

## ✨ Features

### 🔐 User Login

* Personalized login screen
* User session tracking
* Health score system

### 💡 Wellness Assistant

* Daily Health Tips
* Water Intake Guide
* Exercise Recommendations
* Sleep Improvement Tips
* Diet Suggestions
* Motivational Quotes
* Daily Health Challenges

### ⚖️ BMI Calculator

* Calculates Body Mass Index (BMI)
* Displays BMI category
* Provides health advice based on BMI result

### 🩺 Disease Information Database

Detailed information about:

* Diabetes
* Hypertension
* Fever & Flu
* Dengue
* Asthma
* Thyroid Disorders
* Migraine
* Arthritis
* Stress & Anxiety
* Malaria
* Typhoid
* Tuberculosis (TB)
* Jaundice
* Anemia
* Cholesterol
* PCOD / PCOS
* Skin Problems
* Digestive Disorders
* Kidney Issues

### 🚨 Emergency Support

* Ambulance Contacts
* Police Helpline
* Fire Service Contacts
* Medical Helpline Numbers
* Nearby Hospital Information

### 📊 Statistics Tracking

* Total Interactions
* Health Score
* Session Count
* Feature Usage Analytics

### 💾 Chat Management

* Save Chat History
* Clear Chat Window
* Persistent User Statistics using JSON
