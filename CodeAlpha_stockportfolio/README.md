# 📈 Stock Portfolio Tracker

A Python-based Stock Portfolio Tracker built using Tkinter. This application helps users manage stock investments, track portfolio performance, analyze profits/losses, generate reports, and visualize portfolio allocation through charts.

---

## 🚀 Features

### 📊 Portfolio Management
- Add stocks to portfolio
- Remove stocks from portfolio
- Clear entire portfolio
- Auto-calculated investment values

### 💹 Real-Time Portfolio Analysis
- Current market price tracking
- Profit/Loss calculation
- Portfolio return percentage
- Asset allocation percentage

### 📈 Statistics Dashboard
- Total number of stocks
- Best performing stock
- Worst performing stock
- Overall portfolio return

### 📋 Portfolio Details
- Stock quantity
- Average purchase price
- Current market price
- Total invested amount
- Current portfolio value
- Profit/Loss details

### 💾 Data Management
- Save portfolio to CSV file
- Load portfolio from CSV file
- Export detailed TXT report

### 🥧 Data Visualization
- Portfolio allocation pie chart
- Graphical representation of holdings
