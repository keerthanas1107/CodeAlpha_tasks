# 🎮 Hangman Deluxe

A modern Python-based Hangman game built using Tkinter with an attractive GUI, multiple difficulty levels, achievements, leaderboard system, save/resume functionality, sound effects, and customizable themes.

---

## 🚀 Features

### 🎯 Gameplay
- Guess letters to reveal the hidden word
- Multiple categories:
  - Technology
  - Animals
  - Sports
  - Countries
- Three difficulty levels:
  - Easy
  - Medium
  - Hard

### ⏱ Timer System
- Adjustable game timer
- Countdown during gameplay
- Automatic game over when time expires

### 💡 Hint System
- Two hints available per word
- Score penalty for using hints
- Helps players when stuck

### 🏆 Achievements
Unlock achievements such as:

- 🏆 First Win
- 🔥 3-Win Streak
- 💎 Perfect Game
- 🧠 No-Hint Victory
- ⚡ Speed Win

### 📊 Statistics Dashboard
Track:
- Games Played
- Wins
- Losses
- Win Rate
- Best Score
- Current Streak

### 💾 Save & Resume
- Save game progress anytime
- Resume previously saved games
- Automatic game state restoration

### 🏅 Leaderboard
- Top player rankings
- Score tracking
- Win/Loss records
- Category and difficulty tracking

### 🎨 Theme Support
- Dark Theme
- Light Theme
- Modern Dashboard UI

### 🔊 Sound Effects
- Correct guess sound
- Wrong guess sound
- Optional sound settings

## 🎮 How to Play

1. Enter your player name.
2. Choose a category.
3. Select difficulty level.
4. Start the game.
5. Guess letters one at a time.
6. Use hints if necessary.
7. Reveal the word before running out of attempts or time.

---

## 📈 Scoring System

| Action | Points |
|----------|----------|
| Correct Guess | +10 |
| Wrong Guess | -5 |
| Hint Used | -2 |
| Winning Bonus | +50 |

---

## 🏅 Achievement System

| Achievement | Description |
|------------|-------------|
| First Win | Win your first game |
| 3-Win Streak | Win three games consecutively |
| Perfect Game | Win without any wrong guesses |
| No-Hint Win | Win without using hints |
| Speed Win | Win with 30+ seconds remaining |
